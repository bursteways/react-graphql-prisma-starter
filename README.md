# Fullstack React/GQL/Prisma

## Getting started

### Backend setup:
- #### Globally install Prisma
###### packages/api
```bash
$ yarn --global prisma
```
or
```bash
$ npm install -g prisma
```

- #### Grant permission to Prisma
```bash
$ primsa login
``` 
- #### Initialize Prisma
###### packages/api/src
```bash
$ prisma init
```
